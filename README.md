# mars-rover-kata

## requirements
- openjdk15 installed

## running
from the root of the project run ` ./gradlew run --args="initX initY initDirection commands` where:
* `initX` is the initial X coordinate
* `initY` is the initial Y coordinate
* `initDirection` is the initial facing direction (NORTH, EAST, SOUTH, WEST)
* `commands` set of commands to execute (F)

## notes
no area boundaries have been considered
